# Essência Spa

Landing page sofisticada para clínica de estética e spa localizada em **Bragança Paulista**, São Paulo.

---

## Sobre o Projeto

O **Essência Spa** é um site de demonstração desenvolvido para apresentar o potencial de criação de experiências digitais imersivas para o nicho de **bem-estar, estética e spa**.

O projeto combina um design refinado com animações fluidas, criando uma atmosfera de tranquilidade e luxo que reflete a essência de um spa premium.

---

## Tecnologias Utilizadas

| Tecnologia | Descrição |
|-----------|-----------|
| **HTML5** | Estrutura semântica moderna e acessível |
| **Tailwind CSS (CDN)** | Framework utilitário para estilização rápida e responsiva |
| **AOS (Animate On Scroll)** | Animações suaves ao rolar a página |
| **Lucide Icons** | Ícones modernos e leves |
| **Google Fonts** | Tipografia premium com **Lora** (serif) e **Raleway** (sans-serif) |

---

## Seções do Site

1. **Hero** — Imagem zen de impacto com call-to-action e selo de credibilidade
2. **Estatísticas** — Dados de confiança (clientes, anos de experiência, satisfação)
3. **Serviços** — 6 tratamentos com imagens reais, preços e duração
4. **Antes & Depois** — Transformações reais dos tratamentos
5. **Depoimentos** — Avaliações de clientes com fotos e estrelas
6. **Agendamento** — Formulário completo com seleção de horários
7. **Banner CTA** — Vouchers de presente
8. **Footer** — Contato, links rápidos e redes sociais

## Identidade Visual

- **Paleta Principal:** Rosa (`#EC4899`) + Lavanda (`#8B5CF6`)
- **Estilo:** Soft UI / Glassmorphism / Luxo / Relaxamento
- **Sensações:** Tranquilidade, cuidado, sofisticação, feminilidade

---

## Funcionalidades

- Design 100% responsivo (mobile, tablet, desktop)
- Animações ao scroll com AOS
- Efeitos hover interativos nos cards de serviço
- Seleção visual de horários no formulário
- Menu mobile com transição suave
- Navegação fixa com blur ao rolar
- Botão "voltar ao topo"
- Imagens reais do nicho de spa e bem-estar (Unsplash)
